**🎯 Jogo do Número Secreto**

Projeto desenvolvido durante curso da Alura com o objetivo de praticar lógica de programação e desenvolvimento web.

**📚 Sobre o projeto**

O **Jogo do Número Secreto** é uma aplicação simples onde o jogador tenta adivinhar um número gerado aleatoriamente pelo sistema. A cada tentativa, o jogo informa se o número é maior ou menor, até que o jogador acerte.

**🚀 Tecnologias utilizadas**

* JavaScript
* HTML
* CSS
